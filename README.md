Text Search Tool - PowerShell GUI
A lightweight, user-friendly Windows GUI application that searches for text within files of specific extensions and displays clickable results. Built with PowerShell and Windows Forms.

Features
Search any file type - Specify extension (torrent, txt, pdf, etc.) or search all files

Recursive folder search - Option to include subfolders

Case-sensitive/insensitive search toggle

Clickable results - Left-click any result to open the file with its default program

Right-click context menu on results:

Open file

Open containing folder in Explorer

Copy full file path

Copy file name only

Background search - UI stays responsive during search

Cancel button to stop long-running searches

Real-time status updates

Requirements
Windows 7 or later

PowerShell 2.0 or later (comes pre-installed on all Windows versions)

Installation
Download both files:

FileSearchTool.ps1 - The main PowerShell script

FileSearchTool.bat - Launcher (optional, for easy double-click execution)

Place them in any folder (preferably together)
